## 🚀 O que é isso?

Template completo de **README de perfil para GitHub** — o mesmo que uso no meu perfil pessoal.

Com ele, sua página inicial no GitHub vira um cartão de visitas profissional, com badges de skills, estatísticas automáticas e a famosa snake animation de contribuições.

> 💡 Veja o resultado ao vivo: [github.com/Adryannofc](https://github.com/Adryannofc)

---

## 📦 O que está incluído

| Arquivo | Descrição |
|---------|-----------|
| `template/README.md` | Template completo pronto para copiar e adaptar |
| `GUIDE.md` | Guia passo a passo com tudo explicado |
| `.github/workflows/snake.yml` | GitHub Action para gerar a snake animation |

---

## ⚡ Como usar

**1.** Copie o arquivo `template/README.md`

**2.** Crie um repositório no GitHub com o **mesmo nome do seu usuário**

**3.** Cole o conteúdo e substitua os campos marcados com `SEU_`

**4.** Siga o `GUIDE.md` para configurar as estatísticas e a snake

> ⚠️ Leia o `GUIDE.md` antes de começar — especialmente a **Seção 7** sobre a snake animation, que requer configuração extra.
